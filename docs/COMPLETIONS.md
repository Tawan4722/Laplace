# Laplace Shell Completions

Completion scripts live in `scripts/completions/`.

- PowerShell: `. .\scripts\completions\laplace.ps1`
- bash: `source ./scripts/completions/laplace.bash`
- zsh: `source ./scripts/completions/laplace.zsh`
- fish: `source ./scripts/completions/laplace.fish`

These scripts provide command and option completion for the current Laplace CLI surface.
